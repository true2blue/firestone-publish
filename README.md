================== Firestone ===================


**Install & Start**
===================

1.Download and install the mongodb

2.Start the mongodb 
    mongod --dbpath C:/data/db
    mongod --storageEngine inMemory --port 27018 --dbpath C:/data/dbram

3.Run bin/initData.bat

4.Run startup.bat
